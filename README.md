# 🖱 Virtual Mouse using Eye Tracking

This project allows users to control the mouse using eye movement with the help of computer vision.

## 🚀 Features
- Control cursor using eye movement
- Hands-free interaction
- Real-time tracking

## 🛠 Tech Stack
- Python
- OpenCV
- Computer Vision

## ▶️ How to Run
1. Install dependencies
2. Run the Python file
3. Use webcam for tracking
